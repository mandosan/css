Проект сайт новостей
Сделано для практического задания 
~~~~~
Cоздаем два файла: index.html и style.css
В index.html создааем заготовку для нашего сайта новостей, делаем шапку 'новости' и создаем уже сами новости
В файле style.css мы меняем размер картинки нашей новости,размер шрифта, стиль, цвет, и выставляем определенные параметры.
Далее кликаем мышкой два раза по файлу index.html в папке и отрываем наш сайт.
